# J.A.R.V.İ.S Asistanı

Bu proje, Python ve Groq API kullanılarak geliştirilmiş, sesli komutlarla çalışan akıllı bir masaüstü asistanıdır.

## Özellikler
- 🎙️ Sesli komutları anlama ve işleme.
- 🧠 Kişisel bilgileri hafızaya kaydetme (`hafiza.txt`).
- 🤖 Yapay zeka ile doğal sohbet.
- ⚙️ Windows üzerinde tek tıkla (`.exe`) çalışma.

## Nasıl Kurulur?
1. txt dosyasındaki kısmından `JARVIS.exe` dosyasını indirin.
2. txt deki siteden key alın
3. Klasör içerisindeki `jarvis_api_key.txt` dosyasına Groq API anahtarınızı yapıştırın veya exeyi açıp kodu yapıştırın
4. jarvis.exe aktif olcaktır
